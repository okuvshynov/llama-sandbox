This Java program implements a **Least Recently Used (LRU) Cache**.

An LRU cache is a data structure that stores a fixed number of items. When the cache reaches its capacity and a new item needs to be added, it automatically removes the item that hasn't been accessed (read or written) for the longest time.

### 1. What the Program Does

The program defines a generic class `LRUCache<K, V>` that supports two primary operations in **O(1)** (constant) time complexity:
*   **`get(key)`**: Retrieves the value associated with a key. If the key exists, it marks the item as "recently used" by moving it to the front of the list. If the key doesn't exist, it returns `null`.
*   **`put(key, value)`**: Inserts or updates a key-value pair.
    *   If the key exists, it updates the value and moves the item to the front.
    *   If the key is new and the cache is full, it evicts the **least recently used** item (the one at the very end of the list) before inserting the new one.
    *   If the key is new and there is space, it simply adds the item to the front.
