### 4. **`run()` Uses `Date.now()` Without Reset**
```javascript

  }

```
**Issue:** `startTime` is not reset between iterations. This means `wait` becomes negative for subsequent events.

**Fix:** Reset `startTime` to `Date.now()` in each loop iteration.
