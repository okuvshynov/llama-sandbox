This C++ program implements a Huffman coding algorithm to compress text using variable-length binary codes based on character frequencies. Below is a detailed breakdown of its functionality and a time complexity analysis.

## 1. Program Functionality

### Objective
The program builds a Huffman tree from the frequency of characters in a given text, assigns binary codes to each unique character, and uses those codes to create an encoded version of the original text.

### Key Steps

1. **Frequency Counting (`buildTree`)**
   - Counts the occurrences of each character in the input string.
   - Uses a min-heap (priority queue) to merge the least frequent characters first.

2. **Tree Construction (`buildTree`)**
   - Merges two least frequent nodes at each step, creating a new parent node with increased frequency.
   - This continues until only one root node remains.

3. **Code Assignment (`buildCodes`)**
   - Assigns binary codes (0/1) to each leaf node in the Huffman tree.
   - Uses depth-first traversal to assign codes based on the tree structure.

4. **Encoding/Decoding (`encode`)**
   - Encodes each character using its assigned code.
   - Returns the encoded string for transmission or storage.

5. **Output (`main`)**
   - Prints the Huffman codes and the ratio of encoded to original text size.
   - Displays the original and encoded text length in bits.

  - Each merge is O(log n).
