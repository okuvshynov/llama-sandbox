### 1. `compress(input: &[u8]) -> Vec<(u8, u16)>`
This function compresses a byte slice into a vector of (byte, count) pairs. It uses a sliding window approach to identify consecutive repeated bytes.

**Steps:**
1. **Initialization:**
   - Creates an empty `Vec<(u8, u16)>` called `result`.
   - Starts with the first byte of the input (`current`) and an initial count of 1.
2. **Iterating:**
   - For each byte after the first:
     - If the byte matches the current one and the count is less than `u16::MAX`, increment the count.
     - Otherwise, push the previous byte and its count into `result`, reset `current` and `count` to the new byte.
3. **Final Entry:**
   - After the loop, the last byte and its count are pushed into `result`.
4. **Return:**
   - Returns the compressed data.

**Example:**
For input `b"AAABBBCCCCDDDDDEEEEE"` → `[(b' A', 3), (b' B', 3), (b' C', 3), (b' D', 4), (b' E', 5)]`

### 2. `decompress(data: &[(u8, u16)]) -> Vec<u8>`
This function expands a compressed byte slice back into its original byte form.

2. **Iterating:**
   - For each `(byte, count)` pair:
     - Pushes the byte `count` times into the result vector.
3. **Return:**
   - Returns the reconstructed byte sequence.

**Example:**
For input `[(b' A', 3), (b' B', 3), (b' C', 3), (b' D', 4), (b' E', 5)]` → `"AAABBBCCCCDDDDDEEEEE"`
