This Java program implements a **Least Recently Used (LRU) Cache**.

An LRU Cache is a data structure that stores a fixed number of key-value pairs. When the cache reaches its maximum capacity (`capacity`), it automatically removes the item that has not been accessed (via `get` or `put`) for the longest time to make room for new items.
