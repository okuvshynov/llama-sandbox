# Huffman Encoding Program Analysis

## Overview

This C++ program implements the **Huffman encoding** algorithm, a lossless data compression technique that assigns variable-length binary codes to characters based on their frequency. More frequent characters receive shorter codes, reducing the total number of bits needed to represent the input text.

---

## Program Functionality

### 1. **Node Structure**
Represents each node in the Huffman tree:
```cpp
struct Node {
    char ch;        // Character (for leaf nodes)
    int freq;       // Frequency count
    Node* left;     // Left child
    Node* right;    // Right child
};
```

### 2. **buildTree()**
Constructs the Huffman tree:
- Counts character frequencies: `O(n)`
- Initializes a min-priority queue with character nodes: `O(k log k)`
- Iteratively merges the two lowest-frequency nodes: `O(k log k)`
- **Total:** `O(n + k log k)`

### 3. **buildCodes()**
Traverses the tree to generate binary codes:
- Assigns `'0'` for left branches, `'1'` for right
- Leaf nodes store their character's final code
- **Total:** `O(k)`

### 4. **encode()**
Converts input text to a binary string:
- Replaces each character with its Huffman code
- **Total:** `O(n × L)`, where `L` is average code length

### 5. **main()**
- Builds the Huffman tree and code map
- Sorts and displays codes by length
- Compares original and encoded bit sizes
- Reports compression ratio
- **Total:** `O(k log k)`

✅ Efficient compression for texts with uneven character frequencies

## Weaknesses

❌ **Memory leaks** — dynamically allocated `Node` objects are never deleted  
❌ **No error handling** — crashes if character not found in `codes` map  
❌ **Inefficient bit storage** — encodes as string instead of actual bits  
❌ **Not thread-safe** — global state in `main()`  
❌ **No decompression** — only encoding is implemented  

---

## Recommendations for Improvement

1. **Memory Management**
   ```cpp
   void deleteTree(Node* node) {
       if (!node) return;
       deleteTree(node->left);
       deleteTree(node->right);
       delete node;
   }
   // Call deleteTree(root) before program exit
   ```

2. **Bit-Level Encoding**
   Use `std::vector<bool>` or bit manipulation to reduce storage overhead.

3. **Robustness**
   Add checks for missing characters in the `codes` map.

4. **Add Decompression**
   Implement a decoder using the Huffman tree to reconstruct the original text.
