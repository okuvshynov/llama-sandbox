# Bug Analysis

## The Bug

The primary issue in this program is **dead code**: the `merge_sorted_lists` function is defined but **never used** anywhere in the program.

Additionally, there's an **efficiency bug**: the `running_median` function re-sorts the entire window on every iteration, which is computationally expensive.

## Why This Is a Problem

1. **Dead Code**: Unused functions increase code complexity and can mislead future maintainers about the program's intended logic.
2. **Performance Issue**: Sorting the window on every iteration results in O(n × k log k) time complexity, where `k` is the window size. A more efficient approach would maintain the window in sorted order, reducing complexity to O(n × k).
