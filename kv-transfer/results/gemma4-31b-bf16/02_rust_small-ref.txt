### 1. The `compress` function
This function takes a slice of bytes and shrinks it by counting consecutive identical characters.
* **How it works:** It iterates through the input. If the current byte is the same as the previous one, it increments a `count`. If the byte changes (or the count reaches the maximum limit of a `u16` integer), it saves the character and its count as a tuple `(u8, u16)` into a vector.
* **Example:** If the input is `"AAABBB"`, it returns `[('A', 3), ('B', 3)]`.

### 2. The `decompress` function
This function performs the exact opposite of the `compress` function.
* **How it works:** It takes the list of tuples `(byte, count)` and uses a nested loop. The outer loop iterates through the tuples, and the inner loop pushes the specific `byte` into a vector `count` times.
* **Example:** If the input is `[('A', 3), ('B', 3)]`, it expands it back to `"AAABBB"`.

### 3. The `analyze` function
This function performs a **frequency analysis** on the data.
* **How it works:** It uses a `HashMap` where the **key** is the byte and the **value** is how many times that byte appears in the entire input (regardless of whether they are consecutive).
* **Example:** For `"AABBA"`, the map would be `{'A': 3, 'B': 2}`.

### 4. The `main` function (The Execution Flow)
The `main` function puts everything together using the sample data `b"AAABBBCCCCDDDDDEEEEE"`.
